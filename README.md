# Hapi-fresh-start

Hapijs starter kit.

## Install

```bash
git clone https://github.com/sitrakay/hapi-fresh-start my-hapi-project
cd my-hapi-project
npm install
npm start
```
You may now visit `http://localhost:4000` in your browser to view your app.
